Connectify

Connectify is a comprehensive social networking platform developed to help users build meaningful connections. Designed with both a robust backend using Node.js and a dynamic frontend in React, it offers features such as real-time messaging, customizable profiles, and privacy controls. This project showcases my full-stack skills in creating scalable, user-centric applications with a focus on performance and security.

EventEase

EventEase is an event management application that simplifies event organization for individuals and businesses. Built using a MERN stack (MongoDB, Express.js, React, and Node.js), it allows users to create, customize, and manage event details, with options for guest invites, ticketing, and live updates. This project highlights my ability to build full-stack applications that deliver a smooth, responsive user experience.

ShopSmart

ShopSmart is a powerful e-commerce platform designed to help small businesses manage their online stores with ease. Utilizing Django for the backend and React for the frontend, ShopSmart provides inventory tracking, order management, and secure payment options. Through this project, I demonstrated my skills in building feature-rich, full-stack applications with an emphasis on data integrity and secure transactions.
